# Agent skills for the CRAIG RAG/MCP

Claude Code, Copilot CLI and others support use of SKILLS.md files. 

The software-auditor skill included is meant as an example utilizing the CRAIG mcp during software auditing. It was written with Copilot CLI in mind, but can easily be converted to use with Claude Code

## HowTo
In the directory you start copilot cli, create directory .github and copy software-auditor into it. Restart copilot. Check out the SKILLS.md file to see triggers.
